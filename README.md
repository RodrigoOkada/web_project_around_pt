# Tripleten web_project_around

## Visão Geral do Projeto

Around é uma aplicação web interativa de galeria de fotos que permite aos usuários editar seu perfil, adicionar novos locais com imagens, curtir fotos e deletar cartões. O projeto demonstra boas práticas de desenvolvimento web, incluindo validação de formulários, manipulação do DOM e gerenciamento de eventos.

## Funcionalidades

- **Edição de Perfil:** Editar nome e profissão do usuário através de um formulário modal com validação em tempo real.
- **Adição de Cartões:** Adicionar novos locais com título e URL de imagem através de um formulário validado.
- **Visualização de Imagem:** Ampliar imagens em um modal com título do local.
- **Curtir Fotos:** Alternar o estado de curtida de cada cartão.
- **Deletar Cartões:** Remover cartões da galeria.
- **Fechamento de Modais:** Fechar popups ao clicar na sobreposição ou pressionar a tecla Esc.
- **Validação de Formulários:** Validação universal com mensagens de erro em tempo real.

## Tecnologias Utilizadas

- HTML5: Estrutura semântica da página.
- CSS3: Estilização da interface. O projeto segue a metodologia BEM (Block, Element, Modifier) para a nomenclatura de classes, garantindo modularidade e escalabilidade.
- JavaScript (Vanilla JS): Lógica de interatividade, validação de formulários e manipulação do DOM.
